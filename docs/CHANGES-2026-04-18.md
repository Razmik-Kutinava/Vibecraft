### 2. Шапка — фиксация высоты

**Проблема:** при изменении размера логотипа высота шапки растягивалась из-за `align-items: flex-start`, вертикальных padding и отсутствия фиксированной высоты.

**Что сделано:**

- Добавлена CSS-переменная `--header-h: 80px`
- `.header` теперь имеет `height: var(--header-h); overflow: hidden` — высота не растягивается
- Убраны вертикальные padding с `.nav-container`
- На десктопе `align-items: flex-start` → `align-items: center`, убран `padding-top: 8px` у `.nav-panel`
- `.nav-bar`, `.nav-panel` получили `align-self: center`

### 3. Логотип → текстовый wordmark

**Проблема:** PNG-логотип (1536×1024 RGBA) выглядел мелким при любой высоте, SVG был чёрным (невидим на тёмном фоне).

**Решение:** логотип заменён текстовым wordmark прямо в HTML:

```html
<span class="logo-wordmark">
  <span class="logo-vibe">VIBE</span><span class="logo-craft">CRAFT</span>
</span>
```

- **VIBE** — `#f7f7f7` (белый)
- **CRAFT** — `#a3f559` (неоновый зелёный)
- Шрифт: Space Grotesk, `font-weight: 800`, `font-size: 1.75rem`
- PNG/SVG импорты удалены из компонента

**Попутный баг:** `.logo-link` имел `font-size: 0; line-height: 0` (хак для img-зазора) — убран, иначе текст wordmark был невидим.
