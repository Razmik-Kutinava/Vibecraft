# Деплой статики

Сборка создаёт папку `dist/` — это обычный статический сайт (HTML/CSS/JS).

## Общий порядок

1. Установите `PUBLIC_SITE_URL` в `.env` (или в переменных CI) на URL продакшена.
2. Выполните `npm run build`.
3. Загрузите содержимое `dist/` на хостинг или подключите репозиторий к платформе с авто-сборкой.

## Примеры платформ

- **Vercel / Netlify / Cloudflare Pages** — подключить репозиторий, команда сборки `npm run build`, публикация из `dist/` (уточните в настройках «Publish directory»).
- **Любой VPS / Nginx** — скопировать `dist/` в корень сайта и отдавать как статику.

## Проверка после выкладки

- Открыть `/sitemap-index.xml` и убедиться, что URL с вашим доменом
- Проверить `robots.txt` и при необходимости обновить домен вручную
